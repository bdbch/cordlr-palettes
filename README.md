# cordlr-palettes

> Cordlr palettes plugin

## Installation

```sh
$ cordlr install cordlr-palettes
```

Then add it to your config.

```js
{
  "plugins": [
    "cordlr-palettes"
  ],
}
```

## Usage

`$palettes <hex> <default/pastel/soft/light/hard/pale>`
